Que significa BWAP?

BWAP son as siglas de "Buggy Web Apps for Practice", é dicer, unha pequena recopilación de aplicacións webs vulnerabeis con algunha das fallas web máis destacadas e que segundo a OWASP poderemos atopar con maior frecuencia como XSS (Cross Site Scripting) e SQLi (SQL injection), así como algún pequeno exemplo doutras como o XPATHi, e RFI (Remote File Inclusion), esta última descargábel.
Dentro de cada aplicación poderedes atopar un pequeno panel para 'romper', o seu código fonte para que poidades analizalo e a 'solución', que é o xeito de rompela.
Por último, como nivel final atoparedes o mesmo panel, cunha posíbel solución para evitar este tipo de vulnerabilidades co fin de aprender tanto a explotalas como a arranxalas.
